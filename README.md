# face-checkin
ขั้นตอนแรก นำเข้ารายชื่อในไฟล์ google sheet
เลขที่	รหัสนักเรียน	คำนำหน้า	ชื่อ	นามสกุล	ชั้น	รายวิชาที่เรียน

📖 วิธีการใช้งานระบบ
1. โหมดลงทะเบียน (ทำครั้งแรกครั้งเดียว)
ไปที่แท็บ "ลงทะเบียน"
อนุญาตให้เบราว์เซอร์เข้าถึงกล้องเว็บแคม
เลือกรายชื่อนักเรียนจาก Dropdown (ที่มีสัญลักษณ์ ❌ คือยังไม่มีข้อมูลใบหน้า)
มองที่กล้องให้เห็นใบหน้าชัดเจน ไม่สวมหน้ากากอนามัย หรือแว่นตากันแดด
กดปุ่ม "สแกนเพื่อบันทึกใบหน้า" ระบบจะบันทึกโครงสร้างใบหน้าลงในฐานข้อมูลทันที
2. โหมดเช็คชื่อ (ใช้งานประจำวัน)
ไปที่แท็บ "เช็คชื่อ" ระบบจะเริ่มค้นหาใบหน้าอัตโนมัติ
ให้นักเรียนที่ลงทะเบียนแล้วเดินมาอยู่หน้ากล้อง
หากระบบจดจำใบหน้าได้ จะมีกรอบสีเขียวล้อมรอบใบหน้า พร้อมมี เสียงแจ้งเตือนความสำเร็จ (ติ๊ด)
ข้อมูลการมาเรียนจะถูกส่งไปยัง Google Sheet อัตโนมัติ พร้อมแสดงประวัติการสแกนทางขวามือ
หมายเหตุ: หากสแกนซ้ำภายในเวลาที่กำหนด ระบบจะแจ้งเตือนว่า "สแกนซ้ำ" และไม่ส่งข้อมูลซ้ำซ้อน
3. สรุปรายวัน
ไปที่แท็บ "สรุปรายวัน"
เลือกวันที่ต้องการตรวจสอบการมาเรียน
กด "ค้นหาข้อมูล" เพื่อดูประวัติว่าในวันนั้นมีใครเช็คชื่อเวลาใดบ้าง
